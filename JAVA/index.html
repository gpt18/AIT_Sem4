2.3) Logical operator

Code:

    public class logicalOpr_2 {
        public static void main(String[] args) {
            System.out.println((6>4)&&(9>6));
            System.out.println((6>3)&&(9<6));
            System.out.println((6<3)||(8>5));
            System.out.println((6>3)||(8<5));
            System.out.println((6<4)||(8<6));
            System.out.println(!(6==4));
            System.out.println(!(6>4));

        }
    }

Output:

    true
    false
    true
    true
    false
    true
    false

3) Write a java program on loop & switch case.

3.1) while loop

Code:

    public class whileLoop_3 {
        public static void main(String[] args){
            int x=1;

                x++;
            }
        }
    }

Code:

                    public class forLoop_3 {

                            }
                        }
                    }

Output:

3.3) do-while loop

Code:

                    import java.io.*;
                    public class doWhileLoop_3 {
                        public static void main(String[] args) {
                            int x=21;
                            do {
                                System.out.println("Value of x: "+x);
                                x++;
                            } while (x<20);
                        }
                    }

Output:

                    Value of x: 21

3.4) switch case

Code:

                    public class switchCase {
                        public static void main(String[] args) {
                            int day=6;
                            String dayname;
                            switch (day) {
                                case 1: dayname="Monday";
                                    break;
                                case 2: dayname="Tuesday";
                                    break;
                                case 3: dayname="Wednesday";
                                    break;
                                case 4: dayname="Thursday";
                                    break;
                                case 5: dayname="Friday";
                                    break;
                                case 6: dayname="Saturday";
                                    break;
                                case 7: dayname="Sunday";
                                    break;
                                default: dayname="Invalid Day";
                            }
                            System.out.println(dayname);
                        }
                    }

Output:

                    Saturday
